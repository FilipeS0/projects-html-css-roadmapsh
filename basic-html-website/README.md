# Basic HTML Website

A single-page portfolio website built exclusively with HTML (no CSS or JavaScript) as a learning exercise.

## Description

This project is a personal portfolio website created to practice semantic HTML structure and content organization. The main goal was to focus on using native HTML elements effectively to build a multi-page site showcasing projects, articles, work experience, education, and contact information.

## Instructions

[Instructions here](https://roadmap.sh/projects/basic-html-website).

## My project

You can access my project by clicking [here](https://filipes0.github.io/basic-html-website/).

## Features

-   Multi-page navigation with a consistent header and footer across all pages
-   Structured sections for projects, work experience, education, and reviews
-   Contact page with a functional form
-   Semantic HTML5 tags (`<header>`, `<nav>`, `<section>`, `<main>`, `<footer>`)
-   Basic accessibility considerations with proper heading hierarchy
-   Meta tags for better SEO and social media sharing
-   Favicon configuration

## Getting Started

1. **Clone the repository:**

    ```bash
    git clone https://github.com/FilipeS0/basic-html-website.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd basic-html-website
    ```

3. **Open in your web browser:**
    - Open `index.html` in your preferred web browser
    - Or visit the live site: [https://filipes0.github.io/basic-html-website/](https://filipes0.github.io/basic-html-website/)
